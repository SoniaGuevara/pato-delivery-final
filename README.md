## Cómo subir tus cambios a Git

Si notas que tus cambios no aparecen en el repositorio remoto, revisa los
siguientes pasos básicos:

1. **Verifica el estado del repo**
   ```bash
   git status
   ```
   Así confirmas qué archivos se modificaron o están sin seguimiento.
2. **Añade los archivos que quieras enviar**
   ```bash
   git add .
   ```
   o especifica cada archivo manualmente si no quieres incluir todo.
3. **Crea un commit descriptivo**
   ```bash
   git commit -m "Describe brevemente tus cambios"
   ```
4. **Envía el commit al remoto**
   ```bash
   git push
   ```
   Asegúrate de estar en la rama correcta (por ejemplo, `main`, `work`, etc.).

Si el `push` falla, revisa el mensaje de error: puede pedirte que primero hagas
un `git pull --rebase` o que configures las credenciales remotas.
